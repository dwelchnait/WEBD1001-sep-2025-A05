HTML Errors and Validation

using W3C service site

HTML markup follows specific standards and rules to display correctly. It's common to make syntax or logical errors when writing your code. An HTML validation service will analyze your markup, identify any errors or warnings, and provide feedback to help you understand how to fix the issues.

W3C Validation Service

Using the 'Direct Input' tab within the W3C Validator, see if you can find and correct the error with the following code. Afterwards, validate the entire .html file and correct any other issues.

Visual Studio Code Extension

Another way to validate your code is to add an extension into your VSC. The extension is W3C Web Validator. Once installed, a icon/text will appear in your lower left corner (checkmark W3C validation). Simply open the html file of interest and press the button.

Common Types of HTML Errors:

  • Missing closing tags
  • Incorrect tag usage
  • Missing or incorrect usage of attributes
  • Improper nesting
  • Typos
