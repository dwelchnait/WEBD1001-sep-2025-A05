CSS Font Properties

This page is a practice space for CSS selectors and styling text.

Font Families

To set a different font type on your text, use the font-family property. The browser will only apply a font if it is a web-safe font and is available on the user's computer.

If you want all the fonts on a webpage to have the same font type, it's good practice to place this property on the body and it will be inherited by all elements inside the body.

CSS Font Stack is a good resource for web safe fonts.

Font Size

We can set the font size on any element with the font-size property. This property takes values in measurements:

  • px (pixels)
  • em
  • rem
  • % (percentages)

In this course we will be using pixels as a measurement value.

The browser's default font size is 16px, unless it's a heading.

Font Style, Font Weight, Text Transform, and Text Decoration

Font Style

Used for italic text. The property values are:

  • normal
  • italic
  • oblique

Font Weight

Sets how bold the text is. This has many possible values, such as:

  • normal
  • bold
  • A number value from 100 - 900
  • lighter
  • bolder

Text Transform

Transforms the capitalization of your text. Common values are:

  • uppercase
  • lowercase
  • capitalize

Text Decoration

Sets (or unsets) lines through or under text. There are also newer text decoration properties such as text-decoration-style and text-decoration-color that can add some interesting effects.

For the simple text decoration, you can use values such as:

  • none
  • underline
  • overline
  • line-through

Text Alignment

Controls how text is aligned within its content box. Values are:

  • left
  • right
  • center
  • justify - which is not recommended, as it can leave large and irregular spaces between words
